## 📌 Project Overview

This project performs an in-depth **Exploratory Data Analysis (EDA)** on a large COVID-19 dataset using Python. The goal is to uncover patterns, trends, and insights related to the spread and impact of COVID-19 across different regions and time periods.


---

## 🔍 Key Analysis Performed

### 1. 📊 Data Structure Exploration
- Shape, dtypes, and summary statistics
- Unique values and column inspection

### 2. 🧹 Data Quality Detection
- Missing value analysis and heatmaps
- Outlier detection using IQR and boxplots
- Duplicate record identification

### 3. 🔬 Hypothesis Testing
- Statistical tests to validate observations
- Correlation analysis between key features

### 4. 📈 Visualizations
- Trend analysis over time (line plots)
- Regional comparisons (bar charts, choropleth maps)
- Distribution plots using **Matplotlib** & **Seaborn**
